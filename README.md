# Lakewood Roof Repair - Website Redesign

A modern, conversion-optimized website for Lakewood Roof Repair, a family-owned roofing and restoration company serving Pierce County, WA.

## 🎯 Key Features

### Conversion Optimization
- **Trust signals above the fold**: 5.0 Google rating, BBB A+ accreditation, and family-owned badge prominently displayed
- **Simplified contact form**: Reduced from 10+ fields to just 3 required fields (name, phone, email)
- **Multiple CTAs**: Strategic placement of call-to-action buttons throughout the site
- **24/7 emergency service highlighted**: Clear emergency contact options on every page
- **Social proof**: Customer testimonials featuring real names and projects

### Design Improvements
- **Modern, professional design**: Clean layout with excellent visual hierarchy
- **Mobile-responsive**: Fully optimized for all device sizes
- **Fast loading**: Minimal dependencies, optimized CSS and JavaScript
- **Accessibility**: Semantic HTML, proper ARIA labels, keyboard navigation support
- **Brand consistency**: Professional color scheme (blue primary, accent colors)

## 🔧 Customization

### Updating Contact Information
Edit the following in all HTML files:
- Phone number: Search for `+12535551234` and replace
- Email: Search for `christian@lakewoodroof.com` and replace
- Service area: Update in footer and contact page

### Updating Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #1e40af;
    --secondary-color: #0f172a;
    --accent-color: #f59e0b;
}
```

### Adding Form Backend
The contact form currently logs to console. To connect to a backend:
1. Uncomment the fetch code in `script.js`
2. Update the API endpoint
3. Configure your server to handle form submissions
